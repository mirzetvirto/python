# Tic Tac Toe Game Instructions

## Setup Options
1. **Clone the Repository**: 
   - Use the following command to clone the repository:
   ```bash
   git clone https://github.com/mirzetvirto/python.git
   ```

2. **Install Dependencies**: 
   - Navigate to the project directory and install any required dependencies (if applicable).
   - This project may not require additional packages.

## Gameplay Instructions
1. **Start the Game**: 
   - Run the Python script to start the game:
   ```bash
   python tic_tac_toe.py
   ```

2. **Choose Your Symbol**:
   - Players take turns to choose either 'X' or 'O'.

3. **Make Your Move**:
   - Input the position number (1-9) for where you'd like to place your symbol on the grid:
   ```
   1 | 2 | 3
   -----------
   4 | 5 | 6
   -----------
   7 | 8 | 9
   ```

4. **Winning the Game**:
   - The first player to line up three of their symbols horizontally, vertically, or diagonally wins the game.

5. **Restart the Game**:
   - After the game is finished, you can restart it to play again.

Enjoy playing Tic Tac Toe!
